<h2>🛠️ Installation Steps:</h2>

<p>1. Install "npm" packages:</p>

```
npm install
```

<p>2. Add names that you want to check if its available: (should be lowercase)</p>

```
["adnan" "siddiqui"] > 'names.json'
```

<p>3. Now start scrapping usernames:</p>

```
npm start
```
